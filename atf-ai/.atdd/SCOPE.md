# SCOPE — ATF

## Qué hace ATF
- Ejecutar tests de aceptación Gherkin dentro de Docker (Playwright)
- Actualizar atf-state.json y STATE.md con resultados
- Generar reportes HTML y JSON por sprint
- Escribir feedback.md estructurado cuando un test falla
- Publicarse en PyPI como `atf-framework`

## Qué NO hace ATF
- No genera código
- No planifica sprints
- No toma decisiones de arquitectura
- No instala browsers en el host
